# Movies Browser

## Технологии
- React,React Router
- использование локального хранилища,
- работа с API,
- БЭМ
- Express,
- MongoDB

## Функционал
- регистрация и авторизация пользователя,
- поиск фильма по названию
- сохранение и удаление выбранных фильмов
- валидация форм
- редактирование профиля
